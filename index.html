FOCO DO TEXTO

O foco deste texto é a estrutura THERE IS NOT ANY + substantivo incontável.

NOTAS EXPANDIDAS

Uma das maneiras de falarmos que não há/não tem algo (no sentido de não existe) é usando THERE IS NOT ANY. Atenção: agora estamos focando apenas no uso dessa estrutura com PALAVRAS INCONTÁVEIS. Mas o que são mesmo palavras INCONTÁVEIS? Por exemplo: água. Quanto é uma água? Duas águas? Cinco águas? Mil águas? Não dá para contar, não é mesmo? Podemos contar somente litros de água, copos de água etc. Logo, água é incontável, assim como líquidos em geral. Isso vale também para coisas em pó (farinha, açúcar, sal…) ou pastosas (pensem em cookie dough – massa de biscoito). Dinheiro também é incontável. O que seria um dinheiro? Cem dinheiros? Podemos contar dólares, reais, libras, mas não dinheiro.
